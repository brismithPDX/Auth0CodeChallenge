Minimum Viable product for Auth0 Ops Engineering Interview Assignment

Purpose:
    To take incoming Auth0 supplied employee location data in json formatted text file and create a google maps
    plot with all employee locations.

Usage:
    Execute the following command line in a folder containing the script source file

    "python main.py sourcefile.txt"

Dependencies:
    - gmplot
        Required for Final HTML Map Export and Visualisations
    - googlemaps
        Required for Geocoding incoming employee location data
    - google cloud API key
        Required to use googlemaps API Library components
